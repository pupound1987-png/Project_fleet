# **App Name**: FleetLink

## Core Features:

- Employee Login: Secure login using company email to authenticate employees.
- Vehicle Availability: Display real-time status of vehicles (Available, Booked, In Use, Maintenance).
- Booking System: Allow employees to book vehicles, preventing overlapping bookings with immediate error messages.
- Booking Calendar: Provide a calendar view (daily/weekly/monthly) showing vehicle status with color codes.
- Admin Dashboard: Enable admins to manage vehicles, approve/reject bookings, and view booking reports. AI-powered anomaly detection tool for flagging unusual booking patterns.
- Email Notifications: Send email notifications for new bookings, approvals, and rejections.
- User Role Management: Differentiate access and features based on user roles (Employee/Admin).

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB), reflecting a corporate yet approachable feel.
- Background color: Light gray (#F0F8FF), providing a clean and modern interface.
- Accent color: Subtle lavender (#E6E6FA), for interactive elements and highlights.
- Body and headline font: 'PT Sans' for a modern and readable experience.
- Use consistent and clear icons to represent vehicle types and statuses.
- Responsive layout design to support both desktop and mobile devices, with clear menu navigation.
- Smooth transitions and subtle animations to enhance user interaction and provide feedback.
